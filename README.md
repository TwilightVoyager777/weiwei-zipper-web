# weiwei-zipper-web

This is the Next.js web project for Weiwei Zipper, with the original layout retained and the placeholder template content replaced by the current Chinese business copy.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to preview the site.

Primary content now lives under `src/content/`, so branding and page copy can be updated without hardcoding text inside page components.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to load Geist.
